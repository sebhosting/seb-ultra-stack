## ⚡ Overview
SEB Ultra Stack is the **Ultimate WordPress Multisite Production Stack**, tuned for **speed, security, and scalability**.  
One-command install with full CI/CD and GitHub Pages docs.

---

## 🚀 Features
- 🔥 PHP 8.3 + Nginx + Redis, battle-tested config
- 🛡️ Automatic SSL (Let's Encrypt / Cloudflare DNS)
- 📦 WooCommerce & essential plugin-ready
- 💾 Optimized MariaDB with caching
- 🧰 GitHub Actions CI/CD (lint, docs, release, CodeQL)

---

## 📦 Quick Install
```bash
git clone git@github.com:sebhosting/seb-ultra-stack.git
cd seb-ultra-stack
chmod +x install-seb-stack.sh
./install-seb-stack.sh
```
